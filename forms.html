Вступление в команду

Куратор

Куратор - человек, проводящий интересные игры на нашем сервере.


Администратор

Администратор - человек, помогающий Кураторам проводить игры и оказывающие помощь игрокам во время игры.


Модератор

Модератор - человек, следящий за исполнением правил в наших соц.сетях.


Стример

Станьте лицом нашего проекта, демонстрируя игры на стримах.


Мапперы(Hammer)

Мапперы занимаются созданием карт на Hammer для нашего сервера. Не путайте с декораторами.


3D Модельеры(Source)

Создание 3D-моделей для различных пропов, оружий, персонажей.


Импортёры Playermodels в Source

Импортирование для Source моделей различных форматов и из различных источников.


Художники по артам

Рисование артов для каких-либо особых дат, игр и т.д.


Художники по спрайтам персонажей

Рисование готовых спрайтов в полный рост для судов в стиле Danganronp'ы.


Тестер

Тестер должен ходить на "Dev" игры, где тестируются будущие механики режима, и искать все ошибки(баги), чтобы их потом можно было устранить.
